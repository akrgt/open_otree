インストラクション

最初に，各プレイヤーは他のプレイヤーと2人組になります．
しかし，その相手が誰かはわかりませんし，あなたが誰かも相手にはわかりません．

2人のプレイヤーはそれぞれ異なった役割が割り振られます．
一方は提案者であり，もう一方は応答者です．
役割はランダムに割り振られます．割り振られた役割は次のページに表示されます．

あなた方2人はあわせて，{{ Constants.endowment }}を受け取ります．
この実験は，その全体をどのように分けるかについてのものです．
提案者は応答者に対して，いくら提案者が受け取って，応答者がいくら受け取るかを提案します．
そして，応答者はその提案を受け入れるか，拒否するかを決定します．

提案者の役割

  • 提案者は {{ Constants.endowment }}をどのように2人のプレイヤーで分けるのか， 以下の{{ Constants.offer_choices_count }}個の選択肢の中から決定します．

    1. {{ give }} を応答者に渡す． (提案者の手元に {{ keep }}が残る．)
      2. {% endfor %}

    応答者の役割

    {% if not group.strategy %}

  • 応答者はその提案を受け入れる，もしくは拒否することができます．
    • {% else %}
  • 提案者が提案額を決めている間，応答者は{{ Constants.offer_choices_count }}個の選択肢の中から， どの提案を受け入れて，どの提案を拒否するのか選ぶことができます．
  • 提案者が提案額を決めたあと，応答者の選択にしたがって，提案額を受け入れられたか否かを表示します．
    • {% endif %}

    • もし提案が受け入れられたのであれば，両方のプレイヤーはその通りの利得を受け取ることができます．
    • もし提案が拒否されれば，両方のプレイヤーは{{ Constants.payoff_if_rejected }}を受け取ります．
